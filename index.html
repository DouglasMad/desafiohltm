O Surgimento

(Voltar)

A internet foi criada por pesquisadores como Vint Cerf e Bob Kahn, que desenvolveram o protocolo TCP/IP nos anos 1970, permitindo a comunicação entre redes de computadores. Tim Berners-Lee, cientista britânico, inventou a World Wide Web em 1989 e criou o HTML (HyperText Markup Language), a linguagem de marcação usada para estruturar páginas na web. Berners-Lee também desenvolveu os primeiros navegadores e servidores web, revolucionando a forma como acessamos e compartilhamos informações.

Tags

(Voltar)

As tags de HTML são elementos fundamentais na construção de páginas web, definindo a estrutura e o conteúdo dos documentos. Cada tag possui uma função específica, como 'h1' para títulos, 'p' para parágrafos, 'a' para links. Além disso, tags podem ter atributos que fornecem informações adicionais, como id, class, e style , permitindo maior controle sobre o estilo e comportamento dos elementos. O uso correto e semântico das tags melhora a acessibilidade de um site.

Como Funciona o HTML

(Voltar)

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar e estruturar páginas na web. Funciona por meio de tags, que são elementos de código cercados por colchetes angulares, como 'h1' para títulos ou 'p' para parágrafos. Cada tag tem uma função específica e define como o conteúdo será exibido no navegador.

As tags HTML podem conter atributos, que fornecem informações adicionais sobre os elementos. Por exemplo, a tag 'a' cria um link e pode incluir o atributo href para especificar o URL de destino. A estrutura básica de um documento HTML começa com a tag 'Html' , seguida por 'head' para metadados, e 'body' para o conteúdo visível da página.

Além disso, HTML trabalha em conjunto com CSS (Cascading Style Sheets) e JavaScript para criar páginas web interativas e esteticamente agradáveis. CSS é usado para estilizar e formatar os elementos HTML, enquanto JavaScript adiciona funcionalidades dinâmicas. Essa combinação permite a criação de sites modernos e responsivos, essenciais para a experiência do usuário na internet.
